# @ritou11/thuPhdSc

> Select courses for PhD students in Tsinghua University.
>
> Bachelor students use waiting list, so this is useless for them.

## Usage
1. `yarn`

2. Write `config.json` file:

   ```json
   {
     "username": "username",
     "password": "pwd",
     "SEMESTER": "2018-2019-1",
     "TARGET_KCH": "60240103",
     "TARGET_KXH": "0",
     "DELAY_MS": 2000
   }
   ```

3. `yarn start`
